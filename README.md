# THM-Wr1te-Ups
Welcome! This GitHub repository is a collection of walk-throughs and write-ups of THM rooms.

In my cybersecurity career path, I use this as a platform to share my process, methods used, techniques used for exploitation, and any important discoveries made during the process. Should you find yourself struggling with a particular room, need a new angle on an exploit, or just need a little guidance, I hope my documentation helps you crack the code.

🎯 What's Inside?
Step-by-step walkthroughs: Detailed break down of the CTF rooms, easy, medium, and hard.

Methodology and thought processes: Understanding not only what works but the reasoning for doing so.

Payloads: Commands and scripts I used to accomplish exploitation.

📌 Disclaimer
This documentation is meant to serve as educational purposes only. It is encouraged that you attempt to solve these rooms before using the documentation provided.

Happy hacking!
